Initialize your storefront content

This tool will clone the code and content behind aemshop.net to your own code and content repositories. After the GitHub fork is created, please make sure that you install the AEM Code Sync bot. Then the tool will preview and publish your site to Edge Delivery Services!

Important Notes: This tool will only create one fork per template repository per GitHub user account. If you already have a fork of the Boilerplate, you will need delete it to use this tool.
